Step By Step Checklist For Learning Git and Github

  • Download GitBash
  • Download VS Code
  • Git commands:
    • git init
    • git add [insert item file or directory here]
    • git commit -m "[insert commit message here]"
    • git remote add origin https://github.com/SAMPLE/REPOSITORY.git [create a repository on GitHub first and get the HTTPS code from there]
    • git remote -v [verify new remote]
    • git push or git push origin HEAD [pushes commits to online repository on GitHub]
    • git branch [shows you what branch repository your current init is attached to]
